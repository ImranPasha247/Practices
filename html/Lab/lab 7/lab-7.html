Introduction To CSS

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
There are three ways to insert CSS:

  1. External CSS
  2. Internal CSS
  3. Inline CSS

External CSS

External styles are defined within the <link> element, inside the <head> section of an HTML page. The code of External CSS are written in another file with the .CSS Extention

Internal CSS

Internal styles are defined within the <style> element, inside the <head> section of an HTML page. Usually It is similar as the External one but here we dont have to create other folder. We do the style in the html file. But while working with the biggest project. Then it will be hard to track the changes. Thats why internal CSS are not recommended or not a good practice

Inline CSS

Inline styles are defined within the style attribute of an HTML element. Here, We wrtie the codes in the starting tag of any elements. To style that element we use the style key word and then we wrote the CSS part there. But this one also not recommended or not a good practice but it will help some cases.
